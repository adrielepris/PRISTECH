Como usar e publicar no GitHub Pages

  1. Coloque seus vídeos em /videos (arquivos .mp4).
  2. Troque a logo em /assets/logo.svg pelo arquivo oficial da Pristech (mantenha o mesmo nome do arquivo, ou ajuste no index.html).
  3. Edite o arquivo videos.json e atualize a lista com title, src (ex: videos/seu-video.mp4), poster, category, description, durationSec e updated (AAAA-MM-DD).
  4. Abra o index.html localmente para testar. Dica: use uma extensão de servidor local (Live Server) no VS Code para evitar cache.
  5. Envie tudo para um repositório no GitHub. Em Settings → Pages, publique a partir da branch main e pasta / (root).
  6. Acesse sua URL: https://seuusuario.github.io/nomedorepo/

Dicas

  • Para cada vídeo, crie um poster (thumbnail) em /assets/posters. Há 3 exemplos prontos.
  • Você pode organizar por categorias (ex.: Instalação, Administração, Relatórios). O seletor de filtro aparece automaticamente.
  • O botão “Baixar” faz o download direto do arquivo MP4.
